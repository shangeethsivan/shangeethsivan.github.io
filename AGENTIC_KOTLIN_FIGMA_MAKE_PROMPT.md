## Styling Guide for Figma Make

### Theme
- Dark background
- Neon purple and electric blue accents
- Soft gradient mesh layers
- Animated node graph motifs
- Smooth flowchart-style transitions

### Typography
- Large bold titles
- Minimal bullet points
- Centered punch lines
- One strong statement per slide

### Animation Cues
- System slides: node links animate into connected graphs
- Security slide: terminal-style text reveal for “Prompt = Executable Logic”
- Final mindset slides: calm, slow fade transitions
- Keep motion intentional, never noisy

### Presenter Rhythm
- Leave short pauses after punch lines
- Let single-line slides breathe
- Avoid dense narration while animations are active
